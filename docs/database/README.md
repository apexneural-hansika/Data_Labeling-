# Database Documentation

## Files

- `supabase_setup.md` - Complete setup guide for Supabase with pgvector
- `supabase_migration.sql` - SQL script to create tables and indexes
- `setup_database.py` - Python script to automatically set up the database

## Quick Start

1. Run the SQL migration in Supabase SQL Editor:
   ```sql
   -- Copy contents of supabase_migration.sql
   ```

2. Or use the Python setup script:
   ```bash
   python docs/database/setup_database.py
   ```

3. See `supabase_setup.md` for detailed instructions.
